# KAYTITINGA INTEGRATED SCHOOL WEBSITE

# OPEN YOUR TERMINAL AND INSTALL THE FOLLOWING DEPENDENCIES

```bash
py -m pip install fastapi[standard]
py -m pip install "uvicorn[standard]"
py -m pip install jinja2

# AFTER INSTALLING THE DEPENDENCIES, RUN THE FOLLOWING COMMAND TO START THE SERVER
py -m uvicorn page:app --reload

# OPEN YOUR BROWSER AND GO TO THE FOLLOWING LINK
# hold control (ctrl) KEY AND CLICK THE LINK BELOW
http://127.0.01:8000
```
